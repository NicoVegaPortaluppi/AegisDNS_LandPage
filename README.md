# AegisDNS — Landing Page
The landing page for AegisDNS, a desktop application for DNS management and network protection.

Live Site: [nicovegaportaluppi.github.io/AegisDNS_LandPage](https://nicovegaportaluppi.github.io/AegisDNS_LandPage/)

# About
This repository contains the source code for the AegisDNS promotional landing page. The page showcases the features, download options, and information about the AegisDNS desktop application — a tool designed to give users control over their DNS settings with a focus on security and performance.
